# ระบบติดตามงาน & ให้คุณให้โทษ (PWA)

ระบบมอบหมายงาน ติดตามสถานะ และให้คะแนนสะสม/ใบเตือนพนักงาน พร้อมติดตั้งลงเครื่องได้ (PWA)

## ไฟล์ในโฟลเดอร์นี้
- `index.html` — ตัวแอปหลัก
- `manifest.json` — ข้อมูลแอปสำหรับติดตั้ง (ชื่อ, ไอคอน, สีธีม)
- `sw.js` — Service Worker สำหรับ cache และให้ติดตั้งแอปได้
- `icon-192.png`, `icon-512.png` — ไอคอนแอป

## วิธีอัปขึ้น GitHub Pages

1. สร้าง repository ใหม่บน GitHub (เช่น `task-score-app`)
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้น repository (ต้องอยู่ระดับบนสุด ไม่ใช่ในโฟลเดอร์ย่อย)
3. ไปที่ **Settings → Pages**
4. เลือก Branch เป็น `main` (หรือ branch ที่ใช้) และ Folder เป็น `/ (root)`
5. กด Save แล้วรอ 1-2 นาที จะได้ลิงก์ เช่น
   `https://ชื่อผู้ใช้.github.io/task-score-app/`
6. เปิดลิงก์นั้นบนมือถือ/คอม จะเห็นปุ่ม **"ติดตั้งแอป"** ที่หัวเว็บ กดเพื่อติดตั้งลงเครื่องได้เลย

## เริ่มใช้งานครั้งแรก
1. ไปที่เมนู **"จัดการพนักงาน"** เพิ่มชื่อพนักงานและแผนกก่อน
2. ไปที่ **"มอบหมายงาน"** เพื่อสร้างงานใหม่
3. ใช้เมนู **"ให้คุณให้โทษ"** บันทึกคะแนนพร้อมเหตุผลทุกครั้ง

## หมายเหตุ
- ปุ่มติดตั้งแอปทำงานได้เฉพาะเมื่อเปิดผ่าน **HTTPS** เท่านั้น (GitHub Pages เป็น HTTPS อยู่แล้ว ไม่ต้องตั้งค่าเพิ่ม)
- ข้อมูลเชื่อมต่อกับ Firebase project เดียวกับระบบสต็อก (`stock-69302`) แต่ใช้ collections แยก (`tasks`, `staff`, `scorelog`) ไม่กระทบข้อมูลสต็อกเดิม
- หากต้องการเปลี่ยนชื่อแอป/สีธีม แก้ไขได้ที่ไฟล์ `manifest.json`
